About

This is a cloud based borrowing and lending service. We've focused on the deployment of this web service making sure that the cloud configuration follows the SLOs we agreed upon. We used AWS services to deploy this service using Cognito for users and authentication, s3 for the creation and listing of items, and Amplify for the static web hosting.

This web service is our final project for CMPT 474: Web Systems Architecture
